IMPORTANT: Gemma Model Required

To use this app, you need to download the Gemma 3N E2B model file.

1. Visit: https://huggingface.co/google/gemma-3n-E2B-it-litert-preview
2. Download: gemma-3n-E2B-it-int4.task
3. Place the file in this directory (app/src/main/assets/)

The model file is approximately 300MB and is required for the AI features to work.

Without this model file, the app will build but the AI categorization and SMS parsing features will not function.
